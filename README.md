# Refining the Uncertainty of GPM IMERG Precipitation Product Accounting for the Inherent Error from Rain Gauges Estimations

This repository contains the data processing and analysis scripts for the study "Refining the Uncertainty of GPM IMERG Precipitation Product Accounting for the Inherent Error from Rain Gauges Estimations".

## Repository Structure

- `pre-process.ipynb` and `pre-process-station.ipynb`: Data pre-processing scripts for satellite and rain gauge data
- `Fig1.ipynb` to `Fig10.ipynb`: Scripts for generating figures in the manuscript
- `data/`: Sample raw data files including gridded precipitation data and station observations

## Requirements

Python packages: `numpy`, `pandas`, `matplotlib`, `cartopy`, `scipy`, `xarray`, `rasterio`, `tifffile`, `h5py`
